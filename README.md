# Movie Review & Rating Platform

## Project Description
A web-based platform where users can browse movies, submit reviews, and rate movies. The system stores reviews and ratings and displays them dynamically.

## Features
- View movie details
- Submit reviews
- Rate movies
- Store user feedback
- Responsive UI

## Technologies Used
- Java
- HTML
- CSS
- JavaScript
- MySQL

## Project Structure
movie-review-rating-platform/
├── backend/
├── frontend/
├── database/
├── screenshots/
└── README.md

## How to Run
1. Clone the repository
2. Configure MySQL database
3. Run backend server
4. Open frontend in browser
